********** 微服务框架 **********

* 无状态     - 单个服务实例不存储状态
* 高可用     - 服务实例动态扩缩容
* 高并发     - 单实例空接口ops可达10w(2.0GHz/4P)，多实例*N
* 请求路由   - gate负责将请求转发至服务实例
* 负载均衡   - gate负责请求转发的的负载均衡

* 服务发现   - 基于etcd的服务发现
* 存储自由   - 支持mysql/mongo/redis client库
* 多语言对接 - 多语言clientsdk，支持c++/golang/python等等
* http      - 支持通过http调用服务接口
